# 📝 تطبيق قائمة المهام - To-Do List App

تطبيق ويب متقدم وسهل الاستخدام لإدارة المهام اليومية مع ميزة التخزين المحلي.

- ✅ إضافة مهام جديدة بسهولة
- ✏️ تعديل المهام الموجودة
- 🗑️ حذف المهام
- ✔️ تحديد المهام كمنجزة
- 🏷️ تصنيف المهام حسب الأولوية (منخفضة، متوسطة، عالية)

### 📊 التصفية والعرض
- 🔍 عرض جميع المهام
- 🔵 عرض المهام النشطة فقط
- ✅ عرض المهام المنجزة فقط
- 🎨 ترميز ألواني حسب الأولوية

### 💾 التخزين المحلي
- 🔄 حفظ تلقائي للمهام في Local Storage
- 🔒 البيانات آمنة ومحلية 100%
- 💪 عدم فقدان المهام عند إغلاق المتصفح
- 📥 استيراد وتصدير البيانات

### 📈 الإحصائيات
- 📊 عد المهام الكلي
- 🟢 عد المهام النشطة
- ✅ عد المهام المنجزة
- 🏆 عرض الأولويات

### 🎁 ميزات إضافية

- 🧹 مسح جميع المهام

- 📱 واجهة متجاوبة على جميع الأجهزة

## 🚀 كيفية الاستخدام

### التثبيت
1. استنسخ المستودع أو حمل الملفات
2. افتح ملف `index.html` في المتصفح
3. ابدأ في استخدام التطبيق مباشرة!

### الاستخدام الأساسي

#### إضافة مهمة
1. اكتب المهمة في حقل الإدخال
2. اختر مستوى الأولوية من القائمة المنسدلة
3. اضغط زر "إضافة" أو اضغط Enter

#### تعديل مهمة
1. اضغط على زر "التعديل" (أيقونة القلم)
2. عدّل النص
3. اضغط Enter أو خارج الحقل للحفظ

#### إكمال مهمة
- اضغط على المربع بجانب المهمة لتحديدها كمنجزة

#### حذف مهمة
- اضغط على زر "الحذف" (أيقونة السلة)

#### التصفية
- استخدم أزرار التصفية العلوية لعرض:
  - **الكل**: جميع المهام
  - **النشطة**: المهام غير المنجزة
  - **المنجزة**: المهام المكتملة

#### الإجراءات
- **مسح المهام المنجزة**: حذف جميع المهام المكتملة
- **تحميل البيانات**: تصدير المهام كملف JSON
- **مسح الكل**: حذف جميع المهام (مع تأكيد)

## ⚡ الأداء

- **حجم المشروع**: < 50KB (بدون ضغط)
- **سرعة التحميل**: فورية
- **استهلاك الذاكرة**: منخفض جداً
- **التخزين المحلي**: بدون حد (عادة > 5MB)

## 🔒 الأمان

- 🔐 **آمن 100%**: جميع البيانات محلية
- 🚫 **بدون نقل بيانات**: لا توجد اتصالات خارجية
- ✅ **حماية من XSS**: معالجة آمنة للنصوص
- 🛡️ **بدون تتبع**: لا يتم جمع أي معلومات

## 🐛 إصلاح الأخطاء

### المشكلة: البيانات لا تحفظ
**الحل**: تحقق من:
- تفعيل Local Storage في المتصفح
- عدم تفعيل وضع الخصوصية (Incognito)
- مساحة تخزين كافية في الجهاز

### المشكلة: الواجهة لا تظهر بشكل صحيح
**الحل**:
- امسح ذاكرة التخزين المؤقت (Cache)
- أعد تحميل الصفحة
- استخدم متصفح حديث

## 💡 نصائح للاستخدام

1. **حدد الأولويات بحكمة** - ركز على المهام ذات الأولوية العالية أولاً

3. **احفظ بيانات دورية** - استخدم خاصية التحميل بانتظام
